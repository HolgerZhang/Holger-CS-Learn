# C++实验报告

### 实验三 数据类型和数组

代码文件编码：GBK (source code encoding)

本说明文档编码：GBK (this file encoding)

### 一.   实验目的

通过本次实验要达到如下目的：

1.	在C中数组概念的基础上，掌握C++中向量数据类型的使用方法

2.	在C中字符数组和字符串概念的基础上，掌握C++中string数据类型的使用方法

3.	掌握向量的一些常用操作

### 二.   实验内容

1.	用随机数初始化一个10个元素的数组，每个元素的值都是2位数，请输出该数组元素的最大值、最小值和平均值。

2.	从键盘输入一个不包括空格的仅由英文字母组成的字符串，长度不超过200个字符。统计26个英文字母的使用频率（出现的次数），不区分大小写。最后按使用频率从大到小的顺序输出字母（小写字母）及其使用频率，没有出现的字母不输出。

3.	用手机发短信，一条短信资费为0.1元，但限定一条短信的内容在70个字以内（包括70个字）。如果你一次所发送的短信超过了70个字，则会按照每70个字一条短信的限制把它分割成多条短信发送。假设如下数组存储了一个人一个月发送的每个短信的的分别字数
`int arry [12]={35,146,57,13,224,35,99,68,113,79,88,46}`，
试统计一下他当月短信的总资费。

4.	已知数组  `int arry [10]={13,24,35,46,57,35,79,35,93}`，请将用户输入的一个数字按照从小到大的顺序插在对应的位置后输出整个数组。然后将数组中重复的元素去除后输出。

5.	让用户输入n个整数，存储到一个向量中，然后对该向量的值按逆序重新存放。例如，原来的顺序为8,6,5,4,1。要求改为1,4,5,6,8。n和整数的值请用户输入。

6.	编写程序对12个月的英文单词根据字典序进行从小到大的排序，请使用string向量实现。

7.	手工在程序所在目录下创建1个文本文件Data.txt，该文件中存放了9个int数据，各数据之间以空格间隔。编写1个程序，使用文件流的相关方法顺序完成以下操作：

- 读取文件Data.txt中的所有数据； 
- 将读取得到的数据以二进制的形式存放到程序所在目录下的文件Res.dat中；
- 再次读取Res.dat中的所有数据；
- 并将这些数据以文本文件的形式存放到程序所在目录下的Res.txt文件中，存放时每行2个数据，每个数据占10列，右对齐，左边补`'C'`字符。
